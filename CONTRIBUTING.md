# CONTRIBUTING

## ブランチ運用

```
feature/xxx    新機能
fix/xxx        バグ修正
docs/xxx       ドキュメント
```

---

## コミットメッセージ

形式

```
<type>: <内容>
```

### type

```
feat   新機能
fix    バグ修正
docs   ドキュメント
style  コードフォーマット・見た目のみの変更
chore  ビルド・設定・依存関係などの変更
```

### 例

```text
feat: 地図にスポットのピンを表示
fix: ログイン時のクラッシュを修正
docs: READMEを更新
```
