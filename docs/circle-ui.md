# サークル＆足あとUI仕様

## 概要

メイン画面（地図）にユーザーの現在地を中心とした半径20mのサークルを表示する。サークル内に入っているスポットのサムネイルはデザインが変化し、タップすると足あとを残せる。

## メイン画面（地図）

### サークル
- ユーザーの現在地を中心に半径20mの円を描画
- 半透明の塗り + 境界線（青系、opacity 0.15程度、border付き）
- ユーザーの移動に追従してリアルタイムで動く

### スポット表示（正方形サムネイル）

地図上のスポットはピンではなく、正方形の枠付きサムネイル画像で表示する。

| 状態 | デザイン |
|------|---------|
| サークル外（通常） | 32×32px、黒枠(2px)、角丸4px、影付き |
| サークル内（到達可能） | 38×38px に拡大、太い黒枠(2.5px)、フローティングアニメーション |
| 足あと済み | 32×32px、黒枠、画像の上に半透明オーバーレイ + 足あとスタンプ（少し傾き） |

### タップ時の挙動

| サムネイルの状態 | タップ後 |
|----------------|---------|
| サークル内 | スポット詳細シートが展開（「足あとを残す」ボタン有効） |
| サークル外 | スポット詳細シートが展開（「足あとを残す」ボタン無効 / グレーアウト） |
| 足あと済み | スポット詳細シートが展開（ヘッダー画像に足あとスタンプ表示） |

## スポット詳細シート

### シートの表示
- ハーフシートなし。タップで直接フル展開
- シート上部に少し隙間があり、背後の地図がチラ見える
- 角丸20px + 影付き
- スクロール可能

### ヘッダー画像
- 正方形画像が横幅いっぱいに表示
- シート上端の角丸に沿ってクリップ
- 画像タップで地図に戻る（シートを閉じる）

### 「足あとを残す」ボタン

| 条件 | ボタンの状態 |
|------|------------|
| サークル内から遷移 | 有効（黒背景、タップ可能） |
| サークル外から遷移 | 無効（グレー背景、「近づくと足あとを残せます」テキスト） |
| 既に足あと済み | ヘッダー画像に足あとスタンプオーバーレイ、ボタン非表示 |

### ボタンタップ後のフロー
1. 「足あとを残す」タップ
2. カメラ画面（正方形撮影）
3. リアクション選択画面
4. 確定 → 足あと完了画面

## 足あと済みの表現

### 地図上
- サムネイル画像の上に半透明の黒オーバーレイ
- 中央に足あとアイコン（白、少し傾き -15deg）

### 詳細シート
- ヘッダーの正方形画像の中央に足あとスタンプ
- 半透明の黒丸（bg-black/60）+ 足あとアイコン（白、傾き）

## フローティングアニメーション仕様

```css
@keyframes floating {
  0%, 100% { transform: translateY(0px); }
  50% { transform: translateY(-4px); }
}
```

サークル内のサムネイルに適用。

## 足あと完了アニメーション

```css
@keyframes stamp-drop {
  0% { transform: scale(2) translateY(-30px); opacity: 0; }
  60% { transform: scale(1.1) translateY(0px); opacity: 1; }
  80% { transform: scale(0.95) translateY(2px); opacity: 1; }
  100% { transform: scale(1) translateY(0px); opacity: 1; }
}

@keyframes stamp-squish {
  0%, 50% { transform: scale(1); }
  70% { transform: scale(0.95) scaleY(0.92); }
  85% { transform: scale(1.02) scaleY(1.01); }
  100% { transform: scale(1) scaleY(1); }
}
```

- 写真が少し潰れる（stamp-squish）
- 足あとアイコンが上から落ちてバウンド（stamp-drop）

## 現在地の表示

- 青い丸（14px）+ 白い枠線(2.5px)
- 周囲にパルスアニメーション（青の半透明、広がって消える）
- サークルの中心に常に位置する
